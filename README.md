# House Price Prediction Using Machine Learning

This project predicts house sale prices using the Ames Housing dataset. The goal is to analyse housing features and build machine learning models that can accurately estimate house prices.

## Dataset
Source: Kaggle – Ames Housing Dataset

Dataset details:
- Total Records: 1,460 houses
- Features: 81 variables
- Target variable: SalePrice

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Data loading and preprocessing
2. Data cleaning and handling missing values
3. Exploratory Data Analysis (EDA)
4. Feature selection
5. Machine learning model building
6. Model evaluation

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

## Results

Random Forest performed better with:

R² Score: **0.894**

Mean Absolute Error: **$18,279**

## Key Insights
- Overall quality is the most important factor affecting house prices
- Larger houses tend to have higher prices
- Garage size and living area strongly influence house value
